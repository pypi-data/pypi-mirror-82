
dapla_jupyterlab_widgets
=====================================

Version: |release|

A collection of custom Jupyterlab widgets for the Dapla platform


Quickstart
----------

To get started with dapla_jupyterlab_widgets, install with pip::

    pip install dapla_jupyterlab_widgets

or with conda::

    conda install dapla_jupyterlab_widgets
