*************************
Configuration Manager API
*************************
**Module:** ``typhoon.api.configuration_manager``

The Configuration Manager API contains a set of functions to create new and
manipulate existing project configurations.

Example
-------

The following example demonstrates the entire functionality of the configuration
manager API.

In this example, a project is first loaded from an existing *.tcp* file. Then a
configuration is loaded from an existing *.tcfg* file. With handles to these
loaded items, the configuration is changed by selecting the possible options
in the projects configuration. Once this is done, the project configuration can
be generated.
